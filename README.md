# Air Pollution Mortality Rate Analysis

## Project Overview
This project analyzes mortality rates caused by ambient and household air pollution using data analysis and visualization techniques. The objective is to identify trends, compare mortality rates across countries and years, and derive meaningful insights for better understanding of air pollution's impact on public health.

## Problem Statement
Air pollution is one of the leading environmental risk factors responsible for millions of deaths worldwide. This project aims to analyze mortality data associated with ambient and household air pollution and uncover patterns that can support data-driven decision-making.

## Objectives
- Clean and preprocess the dataset.
- Perform Exploratory Data Analysis (EDA).
- Identify trends and patterns in mortality rates.
- Visualize key insights using interactive dashboards.
- Generate recommendations based on the findings.

## Dataset Information
- Dataset: Ambient Household Deaths Dataset
- Format: CSV
- Tools Used: Python, Pandas, NumPy, Power BI

## Data Cleaning Steps
- Checked for missing values.
- Removed duplicate records.
- Renamed columns for consistency.
- Verified data types.
- Handled inconsistent values.
- Created a cleaned dataset for analysis.

## Exploratory Data Analysis
The analysis focuses on:
- Mortality trends over time
- Country-wise comparison of deaths
- Distribution of mortality rates
- Identification of high-risk regions

## Dashboard Features
- Total Deaths KPI
- Country-wise Mortality Analysis
- Trend Analysis by Year
- Interactive Filters and Slicers
- Comparative Visualizations
